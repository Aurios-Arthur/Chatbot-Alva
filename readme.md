📄 Documentação - Chatbot ALVA
📌 Descrição Geral
O Chatbot ALVA é um assistente virtual criado com HTML, CSS e JavaScript que interage com os usuários de forma simples, oferecendo suporte emocional, mensagens motivacionais, meditação guiada e acesso rápido a contatos de emergência.
